// MODULES

24 modules. 4 phases. Zero shortcuts.

Each module is production-grade with full test coverage. Not prototypes. Not proofs of concept. Real solutions backed by research from the best in the industry.

Phase 1

Setup & Security Hardening

The foundation. If setup is hard, users never get to the good stuff.

  • Doctor Framework (Flutter-style health checks)
  • API Key Validation (format + probe + capability)
  • Auto Token Generation (crypto-secure defaults)
  • Network Exposure Audit
  • Supply Chain Integrity (SHA256 + injection detection)
  • Agent Firewall (3-layer LlamaFirewall-style)
Phase 2

Cost Control & Agent Behavior

Your AI should work for you, not burn your wallet.

  • Token Budget Engine (per-session/hour/day caps)
  • Smart Model Router (complexity-based routing)
  • Semantic Cache (hash + trigram similarity)
  • Loop Detection v2 (3-state circuit breaker)
  • Autonomy Governor (progressive trust system)
  • Task Verification (structured completion checks)
Phase 3

Memory, Context & Permissions

An AI that forgets what you told it is an AI that wastes your time.

  • Context Health Monitor (real-time token tracking)
  • Pinned Memory (Mem0-style persistent facts)
  • Structured Compaction (Factory.ai-style JSON)
  • Session Checkpointing (save/resume/time-travel)
  • Permission Profiles (OWASP-aligned presets)
  • Permission Audit Trail (5-layer + secret redaction)
Phase 4

Beginner Experience & Diagnostics

If your grandma can't set it up, your onboarding is broken.

  • Quickstart Command (one-command wizard)
  • Elm-Style Friendly Errors (E001-E010)
  • Agent Templates (6 ready-to-use configs)
  • Interactive Tutorial (7-step walkthrough)
  • Channel Diagnostics (per-channel health checks)
  • Unified Auth Error Handler
